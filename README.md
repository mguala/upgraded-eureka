# Cartones Brillantes TCG

Magic: The Gathering e-commerce with dynamic card catalog from CSV, Gatherer API integration, and responsive design.

## Features

- 📊 Dynamic card catalog from CSV data
- 🔍 Search and filter system
- 🛒 Shopping cart functionality
- 📱 Mobile responsive design
- ♿ Semantic HTML & accessibility

## Tech Stack

- HTML5
- SCSS (SMACSS architecture)
- Vanilla JavaScript
- Papa Parse (CSV processing)
- Gatherer API (card data)

## Local Setup

```bash
# Compile SCSS
sass assets/scss/main-smacss.scss assets/css/main.css

# Serve
python -m http.server 8000
```

Open `http://localhost:8000` in your browser.
